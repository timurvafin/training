# Training Tracker

Силовые тренировки: **мобильное HTML-приложение (Google Apps Script + Sheets) + Claude Code как тренер-аналитик**.

Во время тренировки ты трекаешь подходы в приложении на телефоне; Claude забирает факт из таблицы, применяет правила прогрессии и дописывает следующую неделю.

## Как это работает

```
Claude пишет план (вкладка «План: …») ──► приложение читает план
       ▲                                          │
       │                                          ▼
   /plan-next-week ◄── «Сессии» (факт) ◄── ты трекаешь и завершаешь тренировку
```

1. **План** — вкладка `План: <название>` в таблице (несколько программ можно держать параллельно). Колонки = недели, строки = подходы (вкл. разминочные), ячейка = `повт×вес`.
2. **Приложение** (`app/`) на телефоне: выбираешь **План → Неделю → День**, вбиваешь повторы/вес/RPE по подходам, в конце — «как прошла» (RPE тренировки). Запись уходит во вкладку **«Сессии»**.
3. **Прогрессия** — `/plan-next-week`: Claude читает «Сессии», применяет двойную прогрессию и дописывает новую неделю.

## Компоненты

| | |
|---|---|
| `app/` | Приложение (HTML + Apps Script `Code.gs`/`logic.gs`), тесты, `DATA-CONTRACT.md`, инструкция деплоя `README.md` |
| `.claude/skills/` | `setup` (первичная настройка), `new-plan` (создать программу), `plan-next-week` (прогрессия) |
| `PRIVATE.md` | Профиль, `spreadsheet_id`, URL приложения, секрет (в `.gitignore`) |
| `CLAUDE.md` | Правила тренировок и форматов для ассистента |

## Быстрый старт

1. **Настройка:** запусти `claude` из этой папки → `/setup` (профиль → подключение таблицы → деплой приложения → первый план).
   Деплой приложения вручную/через `clasp` — см. `app/README.md`.
2. **Тренировка:** открой URL приложения на телефоне (на главный экран) → выбери План/Неделю/День → трекай → «Завершить».
3. **Прогрессия:** `/plan-next-week` после недели (или дня) — Claude допишет следующую неделю.
4. **Новая программа:** `/new-plan` (напр. восстановительный цикл после паузы).

## Ключевые принципы
- **Full-body** 2–3×/нед, базовые движения, баланс тяг/жимов.
- **Двойная прогрессия:** сначала повторы (до верха диапазона), потом вес.
- **Разгрузка** каждые 4–6 недель.
- **Безопасность:** жимы/подтягивания не в отказ (RIR 2–3); учёт плеча/поясницы.

## Дисклеймер
Не медицинское приложение и не заменяет врача/тренера. Рекомендации информационные; при боли — прекращать, прогрессировать постепенно.

## Приватность
Персональные данные (имя, возраст, вес, травмы, ID таблицы, URL/секрет) — только в `PRIVATE.md` (в `.gitignore`).
