## Добавление заметок к ячейкам

Заметки добавляются через `batch_update` с операцией `updateCells`:

```json
{
  "updateCells": {
    "rows": [{"values": [{"note": "текст заметки"}]}],
    "fields": "note",
    "range": {
      "sheetId": ВАШ_SHEET_ID,
      "startRowIndex": ROW_INDEX,
      "endRowIndex": ROW_INDEX + 1,
      "startColumnIndex": COL_INDEX,
      "endColumnIndex": COL_INDEX + 1
    }
  }
}
```

**Важно:**
- Индексы **0-based** (строка 1 = startRowIndex 0, колонка A = startColumnIndex 0)
- `fields: "note"` — обновляет только заметку, не трогает значение ячейки
